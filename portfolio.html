Web Development Final Project

Completed fall semester of 2020

Click the photo on the left to view the project. The link at the top right of the screen will take you back.
This was the final for my Web Development class. Everything was written from scratch by me, and it includes 5 pages. The first page is a simple home page that welcomes the reader to the website. The second page is a calculator that uses an API to calculate the result. The third page is a Weather Forecast page which allows the user to enter their Zip code and receive the 7 day forecast for their location. The fourth page is a meant to simulate a weather sensor by allowing the user to enter a Location, Sensor Number, and Value(temperature) and those data values will be added to a sql database. The fifth page displays the contents of the SQL database, and allows the user to select number of results and to sort by location and sensor. The web development class and that final project aswell taught me so much about Javascript and Web Devlopment as a whole. I would not be able to make this current website without them.

Ball Game

Completed January of 2020

This was a side project I worked on during the winter break of 2020-2021 for fun. I used Unity to create the game and coded it in C#. The goal of the game is to get a ball down to the bottom of a hill full of obstacles. There are certain powerups and abilities that you can gain throughout the game which provides some variety. There are also bonus point items you can collect and a leaderboard, so you can compare your high scores with your friends. Making the game I learned so much about C# and game programming as a whole. I've always had a small interest in game development but have never taken a course here at Miami, so making this game taught me a huge amount is a very small amount of time.
