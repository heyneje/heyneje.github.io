Web Development Final Project

Completed fall semester of 2020

This was the final for my Web Development class. Everything was written from scratch by me, and it includes 5 pages. The first page is a simple home page that welcomes the reader to the website. The second page is a calculator that uses an API to calculate the result. The third page is a Weather Forecast page which allows the user to enter their Zip code and receive the 7 day forecast for their location. The fourth page is a meant to simulate a weather sensor by allowing the user to enter a Location, Sensor Number, and Value(temperature) and those data values will be added to a sql database. The fifth page displays the contents of the SQL database, and allows the user to select number of results and to sort by location and sensor. Click the photo on the left to enter the page

Ball Game

Completed January of 2020

This was a side project I worked on during the winter break of 2020-2021 for fun. I used Unity to create the game and coded it in C#. The goal of the game is to get a ball down to the bottom of a hill full of obstacles. There are certain powerups and abilities that you can gain throughout the game which provides some variety. There are also bonus point items you can collect and a leaderboard, so you can compare your high scores with your friends.
